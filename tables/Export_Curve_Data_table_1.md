|    | Curve #     | The curve number will be displayed here.                                                                     |
|---:|:------------|:-------------------------------------------------------------------------------------------------------------|
|  0 | *           | An asterisk denotes that a curve is displayed on the plot                                                    |
|  1 | Mnemonic    | The mnemonic of the curve (usually an abbreviation of the curve name).                                       |
|  2 | Name        | The Full name of the curve.                                                                                  |
|  3 | Step Value  | A decimal number denoting the sampling frequency of the data. ‘Variable’ denotes a mixed sampling frequency. |
|  4 | Start Depth | The start depth of the curve data.                                                                           |
|  5 | End Depth   | End depth of the curve data.                                                                                 |
|  6 | Start Value | The lowest value of curve data.                                                                              |
|  7 | End Value   | The highest value of curve data.                                                                             |
|  8 | File ID     | Name of the original data set defined during loading.                                                        |
