Application Look
The Application Look refers to the GEO interface style.
If you wish to change this, open the Tools tab and click on Application Look - a range of 'Looks' will invoke. Select as desired and the interface will automatically change.
To return to the classic GEO interface (i.e. non ribbons), on the Tools tab, check Classic View.
